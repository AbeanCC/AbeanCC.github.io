MarkDown的介绍

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

相信我，一旦你熟悉了md的使用，你便会对其爱不释手

MarkDown标题

每一级标题对应一个#,最多到6级

MarkDown段落格式

  • 字体
    *斜体*
    **粗体**
    ***粗斜体***
    exp:

  • 我是斜体*

  • 我是粗体*

  • 我是粗斜体*

  • 分隔线
    ---
    就可以分隔了

  • 删除线
    ~~删除内容~~
    exp:
    我被删除了！

  • 下划线
    <u>下划内容</u>
    exp:
    我有鞋子

  • 脚注
    [^要注明的内容]
    [^要注明的内容]: abcd
    exp
    我有帽子啦[^哇]
    [^哇]: I’m hat

MarkDown列表

无序列表

exp

  • I’m First
  • I’m Second
  • I’m Third

有序列表

exp

  1. 第一
  2. 第二
  3. 第三

MarkDown代码区块

  • 用``就可以表示
    exp
    You can really dance!
  • 用```来表示代码块
    exp

MarkDown链接

  • [名字](地址)
    exp
    AbeanCC’s blog

MarkDown图片

  • ![alt 属性文本](图片地址)
    exp

MarkDown表格

  • 表头 表头
    单元格 单元格
    单元格 单元格

exp
| 表头 | 表头 |
| —- | —- |
| 单元格 | 单元格 |
| 单元格 | 单元格 |

值得一提的是，有左对齐，居中和右对齐三种方式

  • =- 左对齐
  • =-= 居中
  • -= 右对齐

当然还有&nbsp;它表示一个空格。
